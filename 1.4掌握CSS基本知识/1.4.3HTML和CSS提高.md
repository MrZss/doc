## 响应式网格（栅格化）布局
使用 HTML 与 CSS 实现类似 BootStrap 的响应式 12 栏网格布局，根据屏幕宽度，元素占的栏数不同。

以下是相关资料

* [BootStrap栅格化布局](http://blog.csdn.net/z742182637/article/details/50466674)

也可以查看bootstrap的源代码找到响应式网格布局的源码自己进行分析

之后课程的手机端一些效果显示就能用到这些知识所以大家务必仔细学习
